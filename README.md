# react-native-filter-camera

Filtered camera for react-native

## Installation

```sh
npm install react-native-filter-camera
```

## Usage

```js
import FilterCamera from "react-native-filter-camera";

// ...

const result = await FilterCamera.multiply(3, 7);
```
